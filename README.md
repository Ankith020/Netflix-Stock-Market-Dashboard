1. # Netflix-Stock-Market-Dashboard
   
Developed an interactive Power BI dashboard to analyze Netflix stock market data from 2018–2023. The dashboard provides insights into stock performance by tracking Open, Close, High, and Low prices across different time periods.

2. # Features

•	Interactive Quarter (Q1–Q4) slicer for dynamic filtering.
•	KPI cards displaying Open, Close, High, and Low stock values.
•	Year-wise comparison using donut, bar, and line charts.
•	Trend analysis by day, month, quarter, and year.
•	Detailed data table for drilling down into stock records.
•	Custom dark-themed dashboard for improved visualization.

3. # Tools Used

•	Power BI
•	Power Query
•	DAX
•	Microsoft Excel (Dataset)

4 .# Key Insights

•	Analysed historical stock price trends over multiple years.
•	Compared stock performance across different time periods.
•	Created an interactive dashboard to support data-driven financial analysis.
